# Qur'an Recitation Evaluation

Audio-based AI system that classifies a recited Qur'an verse into Beginner /
Intermediate / Advanced and reports an explainable Pronunciation / Tajweed /
Fluency score breakdown.

See [`prompts/`](prompts/) for the per-task prompt pack.

## Quickstart

```bash
python3.10 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
